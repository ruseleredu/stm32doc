Você pode usar a API da GitHub para compilar scripts e aplicativos que automatizam processos, promovem integração com GitHub e estendem GitHub. Por exemplo, você pode usar a API para triar problemas, criar um painel de análise ou gerenciar versões.

- [Introdução à API REST](https://docs.github.com/pt/rest/using-the-rest-api/getting-started-with-the-rest-api)
