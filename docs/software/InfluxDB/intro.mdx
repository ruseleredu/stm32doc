## InfluxDB

[InfluxDB](https://www.influxdata.com/) is an open-source time series database developed by InfluxData. It is designed to handle high write and query loads, making it ideal for applications that require real-time analytics and monitoring.

```bash
docker exec -it influxdb-influxdb2-1 bash
```

```bash
influx org create --name ELT73A-LAB07 --description ELT73A-LAB07
```

```bash
influx user create --org ELT73A-LAB07 --name ELT73A-LAB07 --password stm32lab07
```


