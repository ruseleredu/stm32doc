To return a Docusaurus code block with a specific theme like Prism, you should use the ThemeCodeBlock component. 

## LabFromTemplate

### Use
```bash
import LabFromTemplate from "@site/src/components/LabFromTemplate";
```
```bash
<LabFromTemplate labNumber="LAB00" />
```
### Result
<LabFromTemplate labNumber="LAB00" />

## LabFromScratch

### Use
```bash
import LabFromScratch from "@site/src/components/LabFromScratch";
```
```bash
<LabFromScratch labNumber="LAB00" opts="-c"/>
```
### Result
<LabFromScratch labNumber="LAB00" opts="-c opts added"/>
